# Mint

An addon for the [Boze](https://boze.dev) Minecraft client.

Minecraft 26.1.2

## Features

**Modules**
- `AnchorAura` - automatically attacks enemies using respawn anchor explosions
- `AutoTNT` - automatically places TNT around you
- `AutoWither` - automatically builds withers

**HUD**
- `MintWaterMark` - shows the Mint name and version watermark
- `Durability` - shows armor durability for each armor piece

**Extensions**
- `WorldESP`
  - `FuseTimers for TNT`
- `WorldTweaks`
  - `Tnt tint`
  - `Enchant Glint tint and always option`
  - `Damage overlay tint`
- `HandTweaks`
  - `NoInterrupt for swing animation`
- `ViewModel`
  - `FOV for first person`
  - `Main Hand Progress` / `Off Hand Progress`
- `AutoCrystal`
  - `TargetRender`
  - `TargetLine`
- `Options`
  - `InstantPrefix`
  - `CmdHighlight`

**Commands**
- `mint-coinflip` - flip a coin to test your luck
- `mint-modules` - lists all modules of the addon
- `mint-capesources` - lists all cape sources
