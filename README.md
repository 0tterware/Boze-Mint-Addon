# Mint

An addon for the [Boze](https://boze.dev) Minecraft client.

## Features

**Modules**
- `AutoTNT` - automatically places TNT around you
- `AutoWither` - automatically builds withers

**Extensions**
- `WorldESP`
  - `FuseTimers for TNT`
- `WorldTweaks`
  - `Tnt tint`
  - `Enchant Glint tint and always option`
- `ViewModel`
  - `FOV forr first person`
  - `Main Hand Progress` / `Off Hand Progress`
- `AutoCrystal`
  - `TargetRender`
- `Options`
  - `InstantPrefix`
  - `CmdHighlight`

**Commands**
- `mint-coinflip` - flip a coin to test your luck
- `mint-modules` - lists all modules of the addon
- `mint-capesources` - lists all cape sources
